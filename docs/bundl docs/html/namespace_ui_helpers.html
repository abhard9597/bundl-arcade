Detailed Description

Utility helpers and shared UI constants for the bundl. launcher.

This namespace contains reusable values and helper functions used to style, paint, and support the launcher interface. It centralizes colours, panel sizing, asset path lookup, sound playback, and shared drawing logic so the menu and settings screens stay visually consistent.

Author
Aditi Bhardwaj
