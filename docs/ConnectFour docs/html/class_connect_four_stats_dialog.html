Detailed Description

Displays saved Connect Four statistics in a themed dialog window.

The dialog shows three sections of locally stored results and provides a reset button that can clear the saved file through a callback supplied by the menu. The class only handles presentation and relies on the manager for file operations.

Constructs the statistics dialog.

Builds the statistics dialog and populates its controls.

The dialog starts with a supplied snapshot of the current statistics and can optionally receive callbacks for reloading the latest values and for resetting the file when the user presses the reset button.

Parameters
parentParent window that owns this dialog.
statsSnapshotStatistics snapshot used to initialize the labels.
getStatsCallback used to fetch a fresh statistics snapshot after a reset.
onResetStatsCallback used to clear the saved statistics.

The dialog shows one card for each tracked game mode, adds a short note that explains the multiplayer interpretation, and binds the reset button so the labels refresh immediately after a reset.
