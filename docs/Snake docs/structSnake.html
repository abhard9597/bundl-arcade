Detailed Description

Represents the snake and its state.

The Snake structure tracks the snake body, its current movement direction, and whether it should grow on the next move.

void Snake::AdvanceTo ( const Point &  newHead)
inline

Advances the snake to a new head position.

A new head segment is added to the front of the body. If growth is not pending, the tail segment is removed. Otherwise, the snake increases in length by one segment.
