Detailed Description

Provides AI movement logic for an enemy snake.

The EnemyAI class computes a suggested movement direction for the enemy snake, using pathfinding and fallback safety rules.

Chooses the next direction for the enemy snake.

This method is intended to use BFS shortest path logic to move toward food while avoiding both snakes. If no path exists, it falls back to a safe movement that maximizes available space.
