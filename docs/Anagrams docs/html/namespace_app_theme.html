Detailed Description

Provides theme state and colour palette utilities for the application UI.

This namespace groups the palette definition and helper functions used to retrieve the current application theme styling. It is intended to centralize colour choices for backgrounds, cards, text, inputs, buttons, overlays, and shadows.

bool AppTheme::IsDarkMode ( )

Returns whether dark mode is currently enabled.

Returns whether the application is currently using dark mode.

This function checks the BUNDL_DARK_MODE environment variable to determine whether the application should use the dark theme. If the variable is not set, dark mode is enabled by default.
