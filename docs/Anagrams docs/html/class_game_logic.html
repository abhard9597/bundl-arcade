Detailed Description

Manages the game state and logic for the Anagrams game.

This class handles the current rack of letters, the words found so far, the player's score, and the logic for validating submitted words.

SubmitResult GameLogic::TrySubmit ( const std::string & rawInput)

Attempts to submit a word for scoring.

This function takes the raw input from the user, normalizes it, checks if it's valid (only letters, can be formed from the rack, not already found), and if valid, adds it to the found words and updates the score.

Parameters
rawInputThe raw word input from the user (not case-normalized, may contain spaces).
Returns
It returns a SubmitResult indicating whether the submission was accepted and any relevant messages or points gained.
